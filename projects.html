HR Agentic Agent

Completed

Completed V2 capstone workflow for auditable HR screening, combining AI candidate scoring with deterministic validation, logging, shortlist routing, and recruiter notification.

Problem

A single agent can evaluate resumes, but production HR workflows need reliable score checks, audit logs, and controlled email side effects.

Solution

V2 keeps the AI Agent for reasoning, then uses deterministic n8n nodes to recalculate totals, reject unreadable OCR, log every candidate, and email recruiters only on the shortlist branch.

Roadmap

V3 is planned as a multi-agent evidence audit, and V4 is planned as autonomous triage with a dedicated route-planning agent.

Key AI Insight

The capstone pattern is agentic but controlled: AI reasons about fit, code validates the score, and n8n enforces logging plus safe notification routing.

Inventory AI Agent

V2 Verified

Verified capstone workflow for reliable inventory audits, using OpenAI with normalized Google Sheets stock data, structured output, CheckLog audit logging, and bounded Gmail alerts.

Problem

Inventory automation can false-alert when sheet data is blank, malformed, or routed directly through an AI prompt without deterministic checks.

Solution

V2 normalizes Weekly Data rows before the agent, logs every audit to CheckLog, and sends emails only when the fixed low-stock decision matrix requires it.

Roadmap

V3 evolves the same pattern into an OpenAI planner with recent audit history, reorder plan validation, and supplier-facing actions routed through approval.

Key AI Insight

The system is agentic but guarded: OpenAI reasons over clean inventory context, while n8n enforces structured output, logging, validation, and approval boundaries.
