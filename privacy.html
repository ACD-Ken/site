What This Site Collects

This is a personal portfolio site with no advertising, no third-party trackers, and no accounts. Data use is minimal and transparent.

Chat sessions — When you use ACD-Bot, your messages are sent to a secure API hosted on Railway to generate a reply using Claude AI (Anthropic). Conversations are stored temporarily in your browser's sessionStorage so the chat persists as you navigate between pages. This data never leaves your device beyond the API call and is deleted when you close the tab.

Server logs — Each API request logs your IP address, timestamp, and the originating page for security and rate-limiting purposes. Logs are retained temporarily in Railway's infrastructure and are not shared with third parties.

No tracking cookies — This site sets no advertising cookies, analytics cookies, or any cookies that identify you across visits.
