# Modern Portfolio & Tech Guides Website

A complete, production-quality website built with vanilla HTML, CSS, and JavaScript. Features a setup guide, project portfolio, and travel gallery. Can be hosted locally on macOS (tested on macOS Tahoe 26.2 â€” MacBook Air M4, 16GB, 512GB) or deployed via Docker/GitHub Pages.

## Features

- **Three Main Pages**: Home, Setup Guide, Portfolio, Travel Gallery
- **Dark/Light Mode**: CSS-based theme switching with localStorage persistence
- **Responsive Design**: Mobile-first, fully responsive layout
- **Markdown Support**: Setup guide content written in Markdown
- **Dynamic Image Loading**: Travel gallery loads images automatically
- **Portfolio Filtering**: JavaScript-based project filtering
- **Accessibility**: Semantic HTML, ARIA labels, keyboard navigation
- **Performance**: Lazy loading, image optimization, minimal dependencies
