Completed Capstone - V2

HR Agent Auditable Workflow

A completed agentic HR screening workflow that lets AI evaluate a resume, then uses deterministic n8n controls to validate scores, log every candidate, and notify recruiters only when the shortlist rule passes.

Back To Project Card View V3/V4 Roadmap

What Changed From V1

V1 proved the agentic concept by allowing an n8n AI Agent to call tools directly. V2 keeps that reasoning layer, but moves important business rules and side effects into explicit workflow nodes that are easier to test and explain.

AI Evaluation

The AI Agent reads the OCR output and produces structured candidate scores against the hiring role.

Deterministic Guardrails

A validation node recalculates score totals, enforces the decision threshold, and rejects unreadable OCR results.

Controlled Side Effects

Every final result is logged to Google Sheets, while recruiter email runs only on the true shortlist branch.

V2 Workflow Map

[Form: Job Application]
        -> [Extract Form Data]
        -> [Prepare PDF for OCR]
        -> [Mistral OCR]
        -> [AI Agent: HR Screening Evaluation]
              <- [OpenAI Chat Model]
              <- [Structured Output Parser]
              <- [Tool: score_guardrail]
        -> [Validate AI Evaluation]
        -> [Log Candidate Evaluation]
        -> [Shortlist?]
              true -> [Notify Recruiter]
              false -> stop after logging

Why This Is Better For The Capstone

Still Agentic

The workflow uses an AI Agent, OpenAI model, structured parser, and score guardrail tool for role-fit reasoning.

More Defensible

The agent no longer controls irreversible actions. n8n guarantees logging and notification routing.

Easier To Audit

The system separates responsibilities: AI decides, code validates, and workflow branches enforce outcomes.

In V1, the prompt says "always store" and "only email shortlisted candidates." In V2, the workflow guarantees those rules.

V3 And V4 Roadmap

V2 is the completed portfolio milestone. V3 and V4 are planned next versions that expand the same controlled-agent pattern without marking those future versions as completed.

V2 Done

Auditable Agentic Workflow

One evaluator agent scores the candidate. Deterministic validation recalculates totals, logs every candidate, and allows recruiter notification only for shortlists.

V3 Planned

Multi-Agent Evidence Audit

Adds an auditor agent that reviews whether the evaluator agent's scores, decision, and evidence are supported by the OCR text before email is allowed.

V4 Planned

Autonomous Triage Workflow

Adds a triage planner agent that recommends recruiter routing and next action, while deterministic nodes make the final safe route decision.
